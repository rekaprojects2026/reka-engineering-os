# Skill — Scope Guardrails

## MVP includes
- auth
- clients
- intakes
- projects
- task management
- deliverables
- file metadata and Google Drive linking
- dashboard
- search
- activity log
- basic team workload visibility

## Out of scope for MVP
- accounting
- invoicing
- payroll
- HRIS
- mobile native app
- advanced realtime everywhere
- CAD rendering
- client portal
- AI analysis features
- advanced notification systems

## If asked to add future scope early
Politely refuse inside the implementation plan and explain that it belongs to a later stage.
