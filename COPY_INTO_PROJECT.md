# ReKa Engineering OS — Cursor UI/UX Drop-in Pack

Cara pakai:
1. Extract zip ini.
2. Copy semua isi folder ini ke root project `reka-engineering-os` lu.
3. Kalau Windows minta confirm merge folder, pilih **Yes**.
4. File ini hanya **menambah file baru**, tidak dirancang untuk overwrite file coding existing.
5. Di Cursor:
   - buka repo project
   - pastikan Agent dipakai
   - nyalakan Plan Mode
   - `@docs/06_V2_DESIGN_TOKENS_FINAL.md`
   - `@docs/07_V2_CURSOR_UI_UX_RUNBOOK.md`
   - `@sprints/SPRINT_06_CURSOR_UI_UX_INSPECT_PROMPT.md`
   - `@sprints/SPRINT_06_CURSOR_UI_UX_IMPLEMENT_PROMPT.md`
6. Jalankan inspect prompt dulu.
7. Review plan Cursor.
8. Baru jalankan implement prompt.
